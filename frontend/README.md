# Сайт отделочных услуг "ОтделкаПрофи"

Дипломный проект - полнофункциональный сайт для компании по ремонту и отделке помещений.

## Технологии

### Frontend
- React 18
- React Router v6
- Axios
- React Helmet (мета-теги)
- CSS Modules / БЭМ

### Backend
- Node.js
- Express
- SQLite3
- JWT авторизация
- Bcrypt

## Функционал

### Публичная часть
- Главная с калькулятором стоимости
- Каталог услуг с фильтрацией
- Карточки мастеров
- Портфолио с фильтром
- Контакты и форма обратной связи

### Личный кабинет
- Авторизация/регистрация
- Просмотр своих заявок
- Отмена заявок
- Настройки профиля
- История выполненных работ

### Админ-панель
- Управление заявками (изменение статусов)
- Управление услугами (добавление/удаление)
- Статистика
- Отчеты

## Соответствие требованиям на "5"

 Семантическая верстка (header, main, footer, section, article)  
 Адаптивность (320px → 4K, iPhone SE)  
 Дизайн в Figma (продуман для ЦА)  
 Больше 3 страниц (7 страниц)  
 Серверная часть (Node.js + SQLite)  
 Умные селекторы (:nth-child, [class*=""], ::after)  
 Мета-теги + OpenGraph (React Helmet)  
 Своя админка (управление контентом)  
 Функции тестирования (калькулятор ремонта)  
 PageSpeed 80+ (ленивая загрузка, оптимизация)
